# Vue 3 Quiz APP

An swesome multi options Quiz app with VUE js 3


## Installation


1. Clone the project
   ```sh
   git clone https://github.com/jumahmohammadi/vue-quiz-app
   ```
2. go to vue-quiz-app folder/directory


3. run npm install command 
   ```sh
   npm install
   ```
   
4. run npm run dev command
   ```sh
      npm run dev
   ```
